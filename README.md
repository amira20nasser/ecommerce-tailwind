# E-commerce Website with Tailwind CSS
## Features
- Responsive design using Tailwind CSS
- Product listing with images, names, and prices
- Category filtering to easily find products
- Multi-Select Category filter to allow users to select multiple categories at once
- Add to cart functionality
- Cart page to view selected products and total price.
- Each user has their own cart, allowing for personalized shopping experiences.
