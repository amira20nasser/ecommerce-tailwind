# E-commerce Website with Tailwind CSS
## Features
- Responsive Design: Built entirely with Tailwind CSS for mobile-first, responsive layouts.
- Product Listing: Displays products with images, names, and prices.
- Category Filtering: Quickly find products by selecting categories.
- Multi-Select Category Filter: Users can select multiple categories simultaneously.
- Add to Cart Functionality: Easily add products to the shopping cart.
- Cart Page: View selected products and total price in a dedicated cart page.
- User-Specific Cart: Each user has their own cart for personalized shopping experiences.
- Sign In: Authentication implemented via API (dummy API for demo purposes).
